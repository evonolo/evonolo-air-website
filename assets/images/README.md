# Assets Images

Platzieren Sie hier Ihre Bilder für die Website.

## Empfohlene Struktur:

```
assets/images/
├── hero/           # Hero-Section Bilder
├── sections/       # Section-spezifische Bilder
├── gallery/        # Galerie-Bilder
├── team/           # Team-Fotos
└── logos/          # Logos und Branding
```

## Bildoptimierung:

Für beste Performance empfehlen wir:
- **Format**: WebP mit JPEG/PNG Fallback
- **Größe**: Max. 1920px Breite für Hero-Images
- **Kompression**: Tools wie TinyPNG oder ImageOptim nutzen
- **Naming**: Sprechende Namen verwenden (z.B. `hero-background.jpg` statt `IMG_1234.jpg`)

## Placeholder Images:

Für Entwicklung können Sie kostenlose Placeholder-Services nutzen:
- https://picsum.photos/
- https://placehold.co/
- https://unsplash.com/

Beispiel in Markdown:
```markdown
![Alt Text](https://picsum.photos/800/600)
```
